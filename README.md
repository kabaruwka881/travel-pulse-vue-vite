# Travel Pulse

Вёрстка страниц по макетам Figma. Стек: **Vue 3 + Vite**.

## Структура

```
src/
├── assets/
│   ├── images/     # картинки из макетов (PNG)
│   └── icons/      # иконки и логотипы (SVG)
├── components/     # переиспользуемые куски UI
├── pages/          # цельные экраны
├── App.vue         # корневой компонент
├── main.js         # точка входа
└── style.css       # глобально тема + reset + база
```

## Как устроены стили

- **Глобальное**  — в `src/style.css`.
- **Локальное**  — в `<style scoped>` внутри его `.vue`.
- Цвета/шрифты/отступы задаются через CSS переменные в `:root` (`src/style.css`)
 и заполняются точными значениями из макета.
